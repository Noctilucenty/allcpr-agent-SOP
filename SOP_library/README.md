# SOP Library — ALLCPR Smart Manikin

Curated, de-duplicated collection of the SOP documents that are actually usable
(procedures, forms, templates, policies). Pulled from the three overlapping
`SOP/Smart Manikin*` copies, which were byte-identical for every shared file, so
only one clean copy of each was kept. Site-specific media (photos, `.MOV`/`.mp4`
walkthroughs), filled-in assessment forms for individual addresses, data
spreadsheets, proposals, and research writeups were intentionally left out.

## 01_Core_SOPs — the procedures
- **Smart Manikin 专员分点巡检 SOP.docx** — Specialist daily/weekly site-inspection SOP.
- **Smart Manikins 新地点开发 SOP.docx** — BD new-location development SOP (V1.1).
- **Short Term Business Trip SOP.docx** — Travel apply/approve/execute/report/expense SOP (V1.0).
- **Smart Manikin ICPIS 合作分点开发与运营 SOP.docx** — Investor/partner (ICPIS) site development & operations SOP.
- **Smart Manikin New Market Expansion SOP.docx** — Market-expansion SOP (was mis-saved as "Untitled document.docx").

## 02_Forms_and_Templates — reusable blanks
- **Smart Manikin 现场考察表.docx** — Site field assessment form (editable).
- **SMART MANIKIN SITE FIELD ASSESSMENT FORM.pdf** — PDF version of the same form.
- **ICPIS Weekly Site Check Report.docx** — Weekly inspection checklist (bilingual).
- **新点审批 邮件申请模版.docx** — New-site approval email request template.
- **Smart Manikin 选址评分表.xlsm** — Site-selection scoring workbook.
- **分点指示牌 1.0版本.pdf** — Site signage template (v1.0).
